# Game of Block DOJO

Game of blocks DOJO application using [create-t3-app](https://github.com/t3-oss/create-t3-app).

## Develop

### Install

Install dependencies :

```
pnpm i
```

Add and populate .env.local :

```
cp .env.example .env
```

### Run

Start all apps

```
pnpm dev
```

### Test

```
pnpm test
```

### Lint

```
pnpm lint
```

### Format

```
pnpm format
```
